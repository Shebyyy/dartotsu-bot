# Dartotsu Discord Bot

Discord bot for triggering and managing GitHub Actions workflows for the Dartotsu project.

## Features

- Trigger builds with `/build` command
- Check workflow status with `/workflow-status`
- Cancel running workflows with `/cancel-workflow`
- View build logs with `/build-logs`
- Get bot info with `/bot-info`

## Setup

1. Copy `.env.example` to `.env`
2. Fill in your Discord and GitHub tokens
3. Run `npm install`
4. Run `npm start`

## Commands

- `/build platform:android` - Trigger a build
- `/workflow-status` - Check latest workflow status
- `/cancel-workflow` - Cancel running workflow
- `/build-logs` - View workflow logs
- `/bot-info` - Display bot information
